# LeetCode Algorithms (TypeScript)

A personal collection of LeetCode solutions implemented in TypeScript. Each file in the repo corresponds to a single problem and can be run/tested independently.

## Structure
- One file per problem, e.g. `twoSum.ts`, `isPalindrome.ts`, etc.
- No framework required; plain TypeScript/Node.
